# Custom Template generator

Copy all files from [gotest-template](https://github.com/h4ckm03d/dot/tree/master/ide/gotest-template) to `/path/to/template_dir`

Add `go.generateTestsFlags` to the `settings.json`.

`"go.generateTestsFlags": ["-template_dir", "/path/to/template_dir"]`
